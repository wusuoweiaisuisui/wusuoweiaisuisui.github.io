# 策略监控小程序使用说明

## 📱 微信小程序导入步骤

1. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 点击 **"导入项目"**
3. 项目目录选择：`frontend/miniprogram`
4. AppID：选择 **"测试号"**（如果没有）
5. 点击 **"导入"**

## 🌐 服务器地址

小程序已配置为您的公网服务器：
```
http://49.232.79.54:8000/api
```

## 📂 文件结构

```
frontend/miniprogram/
├── app.json          # 小程序配置
└── pages/
    └── index/
        ├── index.js   # 页面逻辑
        ├── index.wxml # 页面结构
        └── index.wxss # 页面样式
```

## ⚠️ 重要说明

**微信小程序要求：**
- **开发阶段**：可以使用HTTP（已配置）
- **正式发布**：需要HTTPS协议和合法域名

## 🎯 修改API地址

如需修改服务器地址，编辑：
`frontend/miniprogram/pages/index/index.js`
修改第一行的 `API_BASE` 即可

## 📊 功能说明

- 实时账户状态和盈亏
- 持仓列表和交易信号
- 策略配置参数
- 每3秒自动刷新
- 下拉刷新
