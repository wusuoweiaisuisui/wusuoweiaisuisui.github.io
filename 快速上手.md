# 🚀 Token中转服务 - 快速上手

## 📌 一句话说明

我们的Token中转服务就像一个**翻译官**：
- 您的应用 → 请求我们的服务 → 我们转发给OpenAI/Anthropic → 返回结果给您

---

## 🎯 最简单的使用方式

### 前提条件
1. ✅ 服务已启动（看到 "Running on http://127.0.0.1:8080"）
2. ✅ 已配置真实的API密钥（在启动脚本里设置）

---

## 💻 使用示例

### 示例1：Python代码

```python
import requests

# 步骤1：配置服务地址
BASE_URL = "http://127.0.0.1:8080"

# 步骤2：发送请求
response = requests.post(
    f"{BASE_URL}/v1/chat/completions",
    headers={
        "Content-Type": "application/json",
        "X-Provider": "openai"  # 选服务商：openai/anthropic/deepseek/zhipu
    },
    json={
        "model": "gpt-3.5-turbo",
        "messages": [
            {"role": "user", "content": "你好，请用一句话介绍自己"}
        ]
    }
)

# 步骤3：获取结果
result = response.json()
print(result)
```

---

### 示例2：在其他软件中使用

如果您的软件支持配置OpenAI API，只需：

| 配置项 | 值 |
|--------|-----|
| API Base | `http://127.0.0.1:8080/v1` |
| API Key | 随便填（我们服务端已配置） |
| 额外Header | `X-Provider: openai` |

---

## 🔧 支持的服务商

在请求头里加 `X-Provider` 来指定：

```
X-Provider: openai
X-Provider: anthropic
X-Provider: deepseek
X-Provider: zhipu
```
