# 虾仔科技 Token中转服务部署指南

## 在腾讯云服务器上部署

### 步骤1：开放8080端口（腾讯云安全组）

1. 登录腾讯云控制台
2. 进入云服务器 → 安全组
3. 找到您服务器对应的安全组
4. 添加入站规则：
   - 协议端口：TCP:8080
   - 来源：0.0.0.0/0
   - 策略：允许

### 步骤2：检查Python环境

在服务器终端执行：

```bash
# 检查Python3是否安装
python3 --version

# 检查pip3是否安装
pip3 --version
```

如果没有安装Python3，先安装：

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip -y

# CentOS/RHEL
sudo yum install python3 python3-pip -y
```

### 步骤3：下载服务脚本

```bash
cd ~
wget https://raw.githubusercontent.com/wusuoweiaisuisui/wusuoweiaisuisui.github.io/main/token_relay_server.py
```

### 步骤4：安装依赖

```bash
pip3 install flask requests
```

### 步骤5：设置API密钥并启动

```bash
# 设置API密钥（替换为您的真实密钥）
export OPENAI_API_KEY="your-openai-key-here"
export ANTHROPIC_API_KEY="your-anthropic-key-here"

# 启动服务
python3 token_relay_server.py
```

### 步骤6：后台运行（可选）

使用nohup后台运行：

```bash
nohup python3 token_relay_server.py > token_relay.log 2>&1 &
```

查看日志：

```bash
tail -f token_relay.log
```

### 步骤7：验证服务

```bash
# 健康检查
curl http://localhost:8080/health

# 或者从外部访问（替换为您的服务器IP）
curl http://49.232.79.54:8080/health
```

## 常见问题

### Q: 端口被占用怎么办？

A: 修改token_relay_server.py中的PORT变量，或者设置环境变量：

```bash
export PORT=8081
python3 token_relay_server.py
```

### Q: 如何停止服务？

A: 查找进程并kill：

```bash
ps aux | grep token_relay_server
kill <进程ID>
```

### Q: 服务重启后需要重新设置API密钥吗？

A: 是的，建议创建一个启动脚本：

```bash
#!/bin/bash
export OPENAI_API_KEY="your-key"
export ANTHROPIC_API_KEY="your-key"
cd ~
nohup python3 token_relay_server.py > token_relay.log 2>&1 &
```

保存为 start_token_relay.sh，然后：

```bash
chmod +x start_token_relay.sh
./start_token_relay.sh
```
