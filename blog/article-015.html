tick数据的坑

每秒几笔甚至几十笔的成交数据，直接拿来分析很容易Garbage In Garbage Out。常见问题：数据缺失（网络抖动）、乱序（交易所重传）、异常成交（错单、操纵）。

tick转OHLCV

def tick_to_ohlcv(tick_df,freq='1min'): ohlcv = pd.DataFrame(); ohlcv['open'] = tick_df['price'].resample(freq).first(); ohlcv['high'] = tick_df['price'].resample(freq).max(); ohlcv['low'] = tick_df['price'].resample(freq).min(); ohlcv['close'] = tick_df['price'].resample(freq).last(); ohlcv['volume'] = tick_df['volume'].resample(freq).sum(); return ohlcv.dropna()
