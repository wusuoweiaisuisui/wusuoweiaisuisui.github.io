为什么收益率不是最重要的指标

先破一个迷信：年化收益率不是评价策略的核心指标。

我见过年化收益率60%的策略，最大回撤达到45%，夏普比率0.8。这种策略在熊市可以让你账户直接腰斩，然后需要翻倍才能回到原点。

反过来，一个年化15%但最大回撤只有5%的策略，夏普比率可能超过2，实际上更值得跑。

指标1：最大回撤（Max Drawdown）

最大回撤是净值从最高点到最低点的跌幅。公式：


MDD = (峰值净值 - 谷值净值) / 峰值净值

我的经验阈值：

  • < 10%：优秀，适合大资金
  • 10% - 20%：可接受，日常监控
  • > 30%：危险，基本不跑

注意：最大回撤通常在压力测试中会更大。如果历史回测最大回撤是15%，实盘压力下可能达到25%。乘以1.5倍做心理准备。

指标2：夏普比率（Sharpe Ratio）

夏普 = (年化收益率 - 无风险利率) / 年化波动率

直白说，就是"每承担一单位风险，能换来多少收益"。

一般来说：

  • < 1：一般，市场上大把替代品
  • 1 - 1.5：良好
  • > 2：优秀，基本可以实盘
  • > 3：怀疑过拟合

我有一次做参数优化，找到一组参数夏普比率4.2，高兴了三天。后来做样本外测试，夏普直接跌到0.6。

夏普比率特别高的策略，第一反应应该是怀疑，不是高兴。

指标3：胜率 vs 盈亏比

这两个指标要一起看，不能单独看。

胜率40%的策略可以盈利，如果平均盈利是平均亏损的3倍。

胜率80%的策略也可以亏损，如果每次亏损金额是盈利的5倍。

Kelly公式可以算最优仓位：


f = (胜率 × 盈亏比 - (1 - 胜率)) / 盈亏比

例如：胜率55%，盈亏比1.5

f = (0.55 × 1.5 - 0.45) / 1.5 = (0.825 - 0.45) / 1.5 = 0.25

最优下注25%的仓位。实际使用时一般取一半（12.5%），更保守。

指标4：交易次数

样本量决定置信度。

一年只有8笔交易的策略，就算胜率100%也没参考价值——样本太少，可能只是运气。

我的标准：至少50笔以上的交易才有统计意义。策略开发阶段尽量跑3-5年的历史数据，确保交易次数充足。

另外，交易次数太多也要注意——日内高频策略手续费会吃掉很大一部分收益，回测时要把手续费算进去。

指标5：Walk-Forward测试（滚动窗口验证）

这是最重要也最容易被忽略的。

普通回测：在全部历史数据上跑，找最优参数。

Walk-Forward：把数据分成若干段，在前几段"训练"参数，在后几段"测试"，依次滚动。

为什么重要？因为普通回测里你可以无限调参，直到找到在这段数据上"最好"的参数——但这些参数在未来大概率不管用。Walk-Forward强制你模拟"用过去的参数交易未来"的场景。

一个参数：如果Walk-Forward测试结果比全期回测差50%以上，这个策略大概率是过拟合的。

实际案例：我是怎么用这5个指标否掉一个策略的

去年开发了一个螺纹钢的RSI均值回归策略：

  • 年化收益：+45%（看起来不错）
  • 最大回撤：-32%（太大了）
  • 夏普比率：0.7（达不到阈值）
  • 交易次数：29次，胜率55%（样本量还行）
  • Walk-Forward：样本外收益跌到+8%（严重过拟合）

四项不达标，一项勉强。直接否掉，不入实盘。

然后换了参数，换了品种（热卷+焦煤套利），这5个指标才都达标。

工具推荐

我现在用Python自建回测框架，指标计算这样写：


def calc_stats(returns):
    annual_return = (1 + returns.mean()) ** 252 - 1
    annual_vol = returns.std() * np.sqrt(252)
    sharpe = (annual_return - 0.02) / annual_vol  # 2%无风险利率
    
    cum_returns = (1 + returns).cumprod()
    rolling_max = cum_returns.expanding().max()
    drawdown = (cum_returns - rolling_max) / rolling_max
    max_drawdown = drawdown.min()
    
    return {
        "年化收益": f"{annual_return:.1%}",
        "年化波动": f"{annual_vol:.1%}",
        "夏普比率": f"{sharpe:.2f}",
        "最大回撤": f"{max_drawdown:.1%}"
    }
