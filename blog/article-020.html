套利策略的核心是找到协整关系稳定的品种配对。

协整检验的原理

两个价格序列P1和P2，如果它们的线性组合P1 - beta*P2是平稳的，就说它们协整。用人话来说：两个品种的价差会围绕某个均值波动，不会越走越远。

Python实现

from statsmodels.tsa.stattools import coint,adfuller; import pandas as pd; def cointegration_test(series1,series2): score,pvalue,_ = coint(series1,series2); print(f"p值: {pvalue:.4f}") if pvalue < 0.05: print("存在协整关系") from statsmodels.api import OLS,add_constant; model = OLS(series1,add_constant(series2)).fit(); beta = model.params[1]; spread = series1 - beta*series2; adf_result = adfuller(spread); return beta,spread

注意事项

协整关系会随时间变化需要定期重新检验；最好用滚动窗口（252天）持续监控；协整不等于因果，相关但不稳定的关系很危险。def rolling_cointegration(series1,series2,window=252): results = []; for i in range(window,len(series1)): s1=series1.iloc[i-window:i]; s2=series2.iloc[i-window:i]; _,pvalue,_ = coint(s1,s2); results.append({'date':series1.index[i],'pvalue':pvalue}); return pd.DataFrame(results)
