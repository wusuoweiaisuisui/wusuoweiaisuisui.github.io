ATR（Average True Range）是量化交易中最被低估的指标。大部分人知道它可以用来设止损，但其实用途远不止于此。

ATR的计算

True Range = max(H-L, abs(H-PC), abs(L-PC)); ATR = MA(True Range,N)

其中PC是前一日收盘价，解决了跳空导致的日内波动计算和趋势行情中的真实波动范围问题。import talib; atr = talib.ATR(high,low,close,timeperiod=14)

用途1：动态止损

def atr_stop_loss(entry_price,atr_value,direction,multiplier=2.0): return entry_price - multiplieratr_value if direction=='long' else entry_price + multiplieratr_value。我用的是2倍ATR止损在螺纹钢上测试效果最好。

用途2：仓位计算

def calc_position(account_equity,risk_pct,atr,entry,contract_size=10): risk_amount = account_equity risk_pct; risk_per_contract = atr 2 * contract_size; return int(risk_amount / risk_per_contract)。单笔亏损不超过账户1%是我的铁律。

用途3：波动率归一化

不同品种波动率差异巨大直接比较没意义。ATR归一化后可以跨品种比较：def atr_normalized(atr,price): return atr / price * 100
