均线是量化交易最基础的工具，但很多人用了很多年也没搞清楚不同均线的区别。

四种均线的数学差异

SMA（简单平均）：平滑但滞后；EMA（指数加权）：近期权重高；WMA（线性加权）：折中方案；GMMA（多组EMA）：识别趋势强度。

import talib; import numpy as np; def four_ma(close,period=20): sma = talib.SMA(close,timeperiod=period); ema = talib.EMA(close,timeperiod=period); weights = np.arange(1,period+1); wma = np.convolve(close,weights/weights.sum(),mode='valid'); return sma,ema,wma

滞后性对比

相同周期下，SMA大于WMA大于EMA大于GMMA(短期组)。但滞后低也意味着噪声大容易被假信号骗。

实战测试数据（螺纹钢2020-2024日线）

SMA(20)：28次交易43%胜率6.2%年化；EMA(20)：31次46%胜率8.7%年化；WMA(20)：30次45%胜率7.9%年化；GMMA：22次51%胜率11.3%年化。结论：GMMA综合表现最好但信号最少。

我的选择

根据策略类型选择：趋势策略用EMA或GMMA（反应快）；套利策略用SMA（过滤噪声）；日内策略用EMA；日线策略用SMA或GMMA。均线没有绝对好坏，关键是品种适配。
