VPIN（Volume-synchronized Probability of Informed Trading）是一个被低估的指标。它能提前识别机构订单的涌入，对期货大波动有预警作用。

VPIN的原理

把成交量分成买量和卖量（通过价格变化方向估算），计算单位时间内信息交易概率。import numpy as np; import pandas as pd; def calculate_vpin(df,bucket_size=5000): df = df.copy(); df['dv'] = np.where(df['close'].diff() > 0,df['volume'],-df['volume']); df['abs_dv'] = df['dv'].abs(); df['cumvol'] = df['volume'].cumsum(); df['bucket'] = df['cumvol'] // bucket_size; vpin_data = df.groupby('bucket').agg({'abs_dv':'sum','volume':'sum'}); vpin_data['vpin'] = vpin_data['abs_dv'] / vpin_data['volume']; return vpin_data['vpin']

实战用法

def vpin_signal(vpin_series,threshold=0.65): current_vpin = vpin_series.iloc[-1]; vpin_mean = vpin_series.mean(); vpin_std = vpin_series.std(); z_score = (current_vpin - vpin_mean) / vpin_std; return z_score > threshold

注意事项

VPIN需要tick级数据分钟级勉强可用；不同品种的VPIN阈值不同需要历史校准；VPIN是先行指标不能单独用必须配合其他信号。我把它作为大盘可能有大动作的预警信号使用，不作为入场依据。
