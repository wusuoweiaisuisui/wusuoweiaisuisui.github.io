很多新手跑完回测，看个总收益率就完事了。这是亏钱的捷径。

夏普比率(Sharpe Ratio)

衡量单位风险获得的超额收益。def sharpe_ratio(returns, risk_free=0.03): excess = returns - risk_free / 252; return np.sqrt(252) * excess.mean() / excess.std()

及格线：大于1.5算可用，大于2.0算优秀。

索提诺比率(Sortino Ratio)

只考虑下行风险：def sortino_ratio(returns, risk_free=0.03): excess = returns - risk_free / 252; downside = excess[excess < 0]; return np.sqrt(252) * excess.mean() / downside.std()

Sortino通常比Sharpe高，因为剔除了上涨那部分噪声。

卡玛比率(Calmar Ratio)

年化收益 / 最大回撤：def calmar_ratio(returns): annual_return = returns.mean() * 252; cumulative = (1 + returns).cumprod(); max_dd = (cumulative / cumulative.cummax() - 1).min(); return annual_return / abs(max_dd)

综合评估框架

年化收益：及格10%良好20%优秀30%+；夏普比率：及格1.0良好1.5优秀2.0+；最大回撤：及格20%良好15%优秀10%；胜率：及格45%良好50%优秀55%+。没有策略能全部达标，关键是知道自己要什么。
