获取期货连续主力合约数据


import akshare as ak
import pandas as pd

# 螺纹钢连续主力合约（RB0）
df = ak.futures_zh_daily_sina(symbol="RB0")
print(df.head())

这个接口返回的是新浪财经的期货日线数据，字段有：日期、开高低收、成交量、持仓量。

注意：symbol格式是 品种代码 + 0（0代表主力）。常用的：

  • `RB0` = 螺纹钢
  • `HC0` = 热轧卷板
  • `I0` = 铁矿石
  • `JM0` = 焦煤
  • `J0` = 焦炭
  • `CU0` = 铜
  • `AL0` = 铝

数据清洗

akshare返回的数据偶尔有问题，要做基本清洗：


def get_futures_data(symbol, start_date=None):
    df = ak.futures_zh_daily_sina(symbol=symbol)
    
    # 日期格式统一
    df["date"] = pd.to_datetime(df["date"])
    df = df.sort_values("date").reset_index(drop=True)
    
    # 去除成交量为0的无效行（节假日有时会有空行）
    df = df[df["volume"] > 0]
    
    # 截取日期范围
    if start_date:
        df = df[df["date"] >= pd.to_datetime(start_date)]
    
    return df

# 获取2020年以来的螺纹钢数据
rb = get_futures_data("RB0", "2020-01-01")
print(f"数据条数: {len(rb)}, 时间范围: {rb['date'].min()} ~ {rb['date'].max()}")

同时获取多个品种

套利策略需要至少两个品种的数据：


symbols = ["RB0", "HC0", "I0", "JM0", "J0"]
data = {}

for sym in symbols:
    try:
        df = get_futures_data(sym, "2020-01-01")
        data[sym] = df.set_index("date")["close"]
        print(f"{sym}: {len(df)}条数据")
    except Exception as e:
        print(f"{sym}获取失败: {e}")

# 合并成一个DataFrame
price_df = pd.DataFrame(data)
print(price_df.tail())

计算价差

有了两个品种的价格，计算价差：


# 方式1：价格差（绝对价差）
price_df["spread_rb_hc"] = price_df["RB0"] - price_df["HC0"]

# 方式2：价格比（相对价差）
price_df["ratio_rb_hc"] = price_df["RB0"] / price_df["HC0"]

对于期货套利，通常用哪种？

  • 同一产业链的品种（如螺纹钢和热卷），规格相近，用**绝对价差**
  • 跨品种但单位不同的（如黄金/白银，单位不同），用**比值**

计算Z-score（信号核心）


def calc_zscore(series, window=20):
    """滚动窗口Z-score"""
    mean = series.rolling(window).mean()
    std = series.rolling(window).std()
    return (series - mean) / std

price_df["z_score"] = calc_zscore(price_df["spread_rb_hc"], window=20)

# 信号判断
price_df["signal"] = 0
price_df.loc[price_df["z_score"] > 2, "signal"] = -1   # 反套
price_df.loc[price_df["z_score"] < -2, "signal"] = 1    # 正套

遇到的接口问题

问题1：futures_zh_daily_sina 有时候返回空DataFrame

原因是新浪数据源不稳定，解决方案：


import time

def get_futures_data_safe(symbol, max_retry=3):
    for i in range(max_retry):
        try:
            df = ak.futures_zh_daily_sina(symbol=symbol)
            if len(df) > 0:
                return df
        except:
            pass
        time.sleep(2)
    return None

问题2：akshare版本不同，某些接口名称不一样

每次升级前先看一下changelog。如果旧代码跑不通，大概率是接口名变了。
