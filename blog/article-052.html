TqBacktest是天勤的官方回测引擎用法和实盘几乎一致。这是目前国内期货回测最接近实盘的工具之一。

基本用法

from tqsdk import TqApi,TqSim,BacktestFinished; from tqsdk.backtest import TqBacktest; api = TqApi(backtest=TqBacktest(start_dt=date(2020,1,1),end_dt=date(2024,12,31)))

TqBacktest vs 自建回测

TqBacktest真实度高（含手续费滑点）但速度较慢需要稳定网络；自建回测速度快灵活性高但数据需自备。

我遇到的问题

1.网络超时：TqBacktest需要云端数据网络不稳定时会超时。2.回测速度慢：5年日线数据要跑10-20分钟。3.数据不一致：云端数据可能和akshare有细微差异。

TqBacktest是实盘前必须跑的一关用它验证过的策略实盘表现差距最小。
