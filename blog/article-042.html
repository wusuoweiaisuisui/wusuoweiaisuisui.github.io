Dual Thrust是我用过的最稳定的日内趋势策略之一。它简单客观不会过拟合。

策略思路

每日开盘前根据前N日的高低点计算一个区间。价格向上突破上轨 --> 做多；价格向下突破下轨 --> 做空；收盘前平仓。

Python实现

import pandas as pd; import numpy as np; def dual_thrust(df,k_up=0.5,k_down=0.5,period=20): df = df.copy(); rolling_high = df['high'].rolling(period).max().shift(1); rolling_low = df['low'].rolling(period).min().shift(1); open_price = df['open']; df['upper'] = open_price + k_up (rolling_high - rolling_low); df['lower'] = open_price - k_down (rolling_low - rolling_high); df['signal'] = 0; df.loc[df['high'] > df['upper'],'signal'] = 1; df.loc[df['low'] < df['lower'],'signal'] = -1; return df

Dual Thrust最适合波动率高的品种（螺纹钢铁矿石）不适合波动低的（国债）。参数不要频繁调整设好之后坚持用。
