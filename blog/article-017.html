RSI是我见过被误用最多的技术指标。网上随便搜RSSI超卖买入能出来几十篇文章，但真的按这个做亏钱的概率超过70%。

RSI的计算公式

RSI = 100 - 100/(1 + RS); RS = N日内上涨幅度均值 / N日内下跌幅度均值。标准参数N=14，是威尔斯-威尔德40年前随手选的，不一定适用于所有品种。

陷阱1：参数依赖

不同品种的超买超卖区间完全不同：import talib; def analyze_rsi(df,periods=[6,14,21]): for p in periods: rsi = talib.RSI(df['close'],timeperiod=p); print(f"RSI({p}): mean={rsi.mean():.1f}")

螺纹钢RSI均值大概在45-55，黄金可能长期在60以上，用同一套阈值必然亏钱。

陷阱2：钝化

强势趋势行情中RSI可以长期维持在70以上，超买之后继续涨。这时超买卖出会踏空整个行情。

正确用法：相对比较

RSI不是用来判断价格高低，而是比较当前强弱与历史强弱：def relative_rsi(close): rsi = talib.RSI(close,timeperiod=14); percentile = (rsi - rsi.rolling(252).min()) / (rsi.rolling(252).max() - rsi.rolling(252).min()); return percentile.iloc[-1]

当分位数大于0.9说明当前是历史最强状态；当分位数小于0.1说明当前极弱可能反弹。
