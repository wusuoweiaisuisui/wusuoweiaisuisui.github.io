资金曲线是策略状态的晴雨表。学会看资金曲线能让你在策略失效前及时发现。

核心指标

def analyze_equity_curve(equity_df): equity = equity_df['equity']; peak = equity.cummax(); current_dd = (equity - peak) / peak; returns = equity.pct_change().dropna(); volatility = returns.std() * np.sqrt(252); return {'current_drawdown':current_dd.iloc[-1],'max_drawdown':current_dd.min(),'annual_volatility':volatility}

策略失效的信号

信号1：回撤幅度超出历史极值（当前回撤大于历史最大回撤的1.5倍）。信号2：胜率系统性下降（滚动胜率突然持续下降）。信号3：收益分布右移（从右偏变成对称或左偏）。

止损规则

回撤大于历史最大值x1.5暂停策略；连续20笔亏损暂停策略；胜率下降超过10%减仓50%；滚动夏普小于0.5退出策略。
