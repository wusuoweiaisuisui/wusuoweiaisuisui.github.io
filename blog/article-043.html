海龟交易法则是量化交易史上最经典的策略之一。虽然已经过去40年但核心思想依然有效。

原始海龟法则

理查德-丹尼斯1983年发明：突破20日最高点 --> 做多；跌破20日最低点 --> 做空；atr止损；金字塔加仓。

TqSdk实现

from tqsdk import TqApi,TqAccount; from tqsdk.tafunc import atr; api = TqApi(); def turtle_trade(api,symbol,period='day'): klines = api.get_kline_serial(symbol,duration_sec=86400000 if period=='day' else 300); high_20 = klines['high'].rolling(20).max(); low_20 = klines['low'].rolling(20).min(); klines['atr'] = atr(klines,20); entry_long = klines['close'] > high_20.shift(1); entry_short = klines['close'] < low_20.shift(1); return entry_long,entry_short
