因子设计思路

好的因子要有：经济逻辑支撑（不是纯数据挖掘）；能在不同品种/周期上复用；因子值分布合理（非极端偏态）。

因子示例：动量-波动率比

def momentum_volatility_ratio(df,mom_period=20,vol_period=20): momentum = df['close'].pct_change(mom_period); volatility = df['close'].pct_change().rolling(vol_period).std() * np.sqrt(252); factor = momentum / volatility; return factor

在backtrader中实现

import backtrader as bt; class MomentumVolFactor(bt.Indicator): lines = ('factor',); params = (('mom_period',20),('vol_period',20)); def __init__(self): mom = self.data.close.pct_change(self.p.mom_period); vol = self.data.close.pct_change().rolling(self.p.vol_period).std(); self.lines.factor = mom / vol

因子正交化

多个因子一起用时需要做正交化避免多重共线性：from sklearn.decomposition import PCA; def orthogonalize(factors_df): factors_clean = factors_df.dropna(); pca = PCA(n_components=None); orthogonal = pca.fit_transform(factors_clean); return pd.DataFrame(orthogonal,index=factors_clean.index)
