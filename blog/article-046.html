止损是趋势策略的生死线。设太宽单笔亏损大设太窄频繁被扫。

固定止损

def fixed_stop(entry_price,direction,stop_pct=0.02): return entry_price (1 - stop_pct) if direction=='long' else entry_price (1 + stop_pct)。优点简单缺点没有考虑市场波动率。

ATR追踪止损

import talib; atr = talib.ATR(high,low,close,timeperiod=14); def atr_stop(entry_price,direction,atr_value,multiplier=2.0): return entry_price - multiplieratr_value if direction=='long' else entry_price + multiplieratr_value

动态调整的追踪止损

def trailing_stop(position,current_price,atr_value,direction): current_stop = position.get('stop',position['entry']); new_stop = max(current_price - 2atr_value,current_stop) if direction=='long' else min(current_price + 2atr_value,current_stop); position['stop'] = new_stop; return new_stop

结论：ATR 2倍止损在胜率和亏损控制上最平衡。
