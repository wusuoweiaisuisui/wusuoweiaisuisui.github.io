为什么选backtrader

相比TqSdk专注国内期货，backtrader是个通用框架，股票、期货、数字货币都能回测。它最大的优点是数据分析能力极强，内置了Sharpe、Drawdown等一堆指标。缺点是数据源要自己处理，实盘对接没有TqSdk方便。我的用法是：回测用backtrader，实盘切换TqSdk。

最小可运行代码

import backtrader as bt

class MyStrategy(bt.Strategy):

def __init__(self):

self.ma = bt.indicators.SMA(self.data.close, period=20)

def next(self):

if self.data.close[0] > self.ma[0] and not self.position: self.buy()

elif self.data.close[0] < self.ma[0] and self.position: self.close()

cerebro = bt.Cerebro()

cerebro.addstrategy(MyStrategy)

data = bt.feeds.YahooFinanceData('BTC-USD', fromdate=datetime(2020,1,1))

cerebro.adddata(data)

cerebro.run()

cerebro.plot()

手续费和滑点

回测如果不设手续费，收益率虚高30%以上：cerebro.broker.setcommission(commission=0.0003); cerebro.broker.set_slippage_perc(0.0005)
